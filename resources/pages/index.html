JLCEDA2KICAD Importer

Install and update the Windows IPC plugin from a custom KiCad repository.
通过自定义 KiCad 仓库安装和更新 Windows IPC 插件。

KiCad 10 / 推荐

https://hkrpt.github.io/JLCEDA2KICAD/v2/repository.json

KiCad 9

https://hkrpt.github.io/JLCEDA2KICAD/v1/repository.json

Install / 安装

  1. Open Plugin and Content Manager / 打开“扩展内容管理器”。
  2. Open Manage, add the matching URL, save, and refresh / 点击“管理”，添加对应 地址，保存并刷新。
  3. Search for JLCEDA2KICAD Importer and install / 搜索并安装插件。
  4. Open a board and click the teal JLC toolbar button; some builds also show Tools → External Plugins / 打开 PCB 后点击顶部青绿色 JLC 图标；部分版本也会 显示“工具 → 外部插件”。

Offline installation: download the PCM ZIP from GitHub Releases and choose Install from File without extracting it. The published ZIP includes plugins/vendor and does not download Python packages on first launch.
离线安装：下载发行页中的 PCM ZIP，不解压，使用“从文件安装”。正式包已经内置 Python 运行依赖，首次启动不会下载 Python 包。
